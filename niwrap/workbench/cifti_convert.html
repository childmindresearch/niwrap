540    """
541    Dump cifti matrix into other formats.
542    
543    This command is used to convert a full CIFTI matrix to/from formats that can
544    be used by programs that don't understand CIFTI. You must specify exactly
545    one of -to-gifti-ext, -from-gifti-ext, -to-nifti, -from-nifti, -to-text, or
546    -from-text.
547    
548    If you want to write an existing CIFTI file with a different CIFTI version,
549    see -file-convert, and its -cifti-version-convert option.
550    
551    If you want part of the CIFTI file as a metric, label, or volume file, see
552    -cifti-separate. If you want to create a CIFTI file from metric and/or
553    volume files, see the -cifti-create-* commands.
554    
555    If you want to import a matrix that is restricted to an ROI, first create a
556    template CIFTI file matching that ROI using a -cifti-create-* command. After
557    importing to CIFTI, you can then expand the file into a standard
558    brainordinates space with -cifti-create-dense-from-template. If you want to
559    export only part of a CIFTI file, first create an roi-restricted CIFTI file
560    with -cifti-restrict-dense-mapping.
561    
562    The -transpose option to -from-gifti-ext is needed if the replacement binary
563    file is in column-major order.
564    
565    The -unit options accept these values:
566

569    METER
